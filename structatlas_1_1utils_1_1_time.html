Detailed Description

Stores all timing data required for animation driven Scenes.

This struct holds the following information:

  • The total time the scene has been running,
  • the time delta between the last tick and the current tick,
  • and the current tick.

This should provide all of the necessary information for most animation algorithms.

Note
The time units used in Atlas (and the ones in GLFW) are seconds.
