Detailed Description

Includes GLM vector headers and typedefs for Vector classes.

This header includes all of the necessary GLM headers to perform all vector operations. As a result, the GLM headers should not be included directly (unless there is a compelling reason to do so). Instead, the user should simply add this line:

#include <atlas/math/Vector.hpp>

and all vector operations will become available.

Note
The typedefs given here are given from a 3D mathematical perspective. As a result, Vector, Point, and Normal all refer to 3D vectors. The reason why they are typedefed differently is to provide syntax clarity as to what each structure is used for.
