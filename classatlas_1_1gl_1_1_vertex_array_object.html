Detailed Description

Wrapper for OpenGL vertex array objects.

This class wraps around the most common vao operations in OpenGL. It also provides automatic debugging for errors from OpenGL. In addition it contains a list of all the vertex attribute array indeces that are used when this vao is bound. That way they can be set once and the class will take care of deleting them when the destructor is called.
