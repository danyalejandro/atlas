Detailed Description

Defines a mathematical ray with an origin and a direction.

This class can be used for ray casting, or any other operation that requires rays. The () operator has been overloaded as a more intuitive way of evaluating the ray at the given value.

Note
The direction given to the ray need not be normalized, and the class does not normalize directions.
