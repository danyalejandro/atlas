Detailed Description

template<typename GenType = float>
class atlas::math::RandomGenerator< GenType >

Defines a simple random number generator.

This class wraps around the new STD <random> functionality by providing a simple way of getting a random number generator running quickly. This is based on the mt19937 generator, which should be sufficient for most applications. The class is templated to allow switching between floats and doubles.
